# Time Zone Map

[https://timezonemap.lepepe.dev/](https://timezonemap.lepepe.dev/)

An interactive world map to explore and convert time zones.

## Features

- **Interactive Map**: Click on any region to see its current time and time zone details.
- **Real-time Clocks**: View your local time, UTC, and the time in the selected zone simultaneously.
- **Time Conversion**: Convert any specific time from your local (or any other) time zone to a destination zone.
- **Multi-language Support**: Easily switch between English and Spanish.
- **Theme Customization**: Dark and light mode support for better accessibility and user preference.
- **City Search**: Quickly find and select time zones by city names.

## Tech Stack

- **React + TypeScript + Vite**
- **Tailwind CSS**
- **Shadcn UI**
- **Luxon** (for date and time manipulation)
- **Lucide React** (for icons)

## Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Run the development server: `pnpm dev`
4. Build for production: `pnpm build`
